BuddyPress Themes
''''''''''''''''

1. Copy the 'buddypress-member' and 'buddypress-home' directories from /buddypress-theme/ into /wp-content/themes/
2. You may need to log in as an administrator and activate the 'buddypress-home' theme in 'Site Admin > Themes'
3. Activate the buddypress-home theme in the design tab for the home blog.
4. You do not need to activate the buddypress-member theme, this will be auto selected for member pages.
5. DO NOT rename the 'buddypress-home' theme (including the folder name and css file theme name)
6. All done.
